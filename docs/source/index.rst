SolrPHPSolariumBundle
=====================
this is the documentation for the `SolrPHPSolariumBundle <https://github.com/solrphp/solarium-bunde>`_.
use this bundle to:

* use the solarium solr client in your symfony application
* configure and manage your solr schema(s)
* configure and manage your solr config(s)
